myFlix Documentation Page

This web application provides users with access to information about different movies, directors and genres. Users are able to sign up, update their personal information and create a list of their favorite movies.

REST API Endpoints

Request HTTP Method URL Endpoint Response
Get list of all movies GET /movies JSON array of movie objects
Get data about a single movie by title GET /movies/:title JSON object with movie details containing title, genre, description, director and year released. Example: Title: "Amelie", Genre: "Romantic Comedy", Director: "Jean-Pierre Jeunet", Description: "Despite being caught in her imaginative world, Amelie, a young waitress, decides to help people find happiness. Her quest to spread joy leads her on a journey where she finds true love.", Year: 2001,
Get data about a genre by name GET /movies/genre/:genreName JSON object with genre details
Get data about a director by name GET /movies/directors/:directorName JSON object with director details
Register new user POST /users JSON object with user details (success message)
Update user info PUT /users/:userId JSON object with updated user details (succes message)
Add movie to favorites POST //users/:userId/favorites JSON object indicating movie added (success message)
Remove movie from favorites DELETE /users/:userId/favorites/:movieId JSON object indicating movie removed (success message)
Deregister user DELETE /users/:userId/ JSON objec indicating user removed (success message)
