myFlix Documentation Page

This web application provides users with access to information about different movies, directors and genres. Users are able to sign up, update their personal information and create a list of their favorite movies.

REST API Endpoints

Request URL Endpoint HTTP Method Requst Body Response Body
Get list of all movies /movies GET None JSON array of movie objects
Get data about a single movie by title /movies/:title GET None JSON object with movie details containing title, genre, description, director and year released. Example:{ Title: "Amelie", Genre: "Romantic Comedy", Director: "Jean-Pierre Jeunet", Description: "Despite being caught in her imaginative world, Amelie, a young waitress, decides to help people find happiness. Her quest to spread joy leads her on a journey where she finds true love.", Year: 2001}
Get data about a genre by name /movies/genre/:genreName GET None JSON object with genre details
Get data about a director by name /movies/directors/:directorName GET None JSON object with director details. Example: { Director: "Christopher Nolan", Bio: "Christopher Nolan, born on July 30, 1970, in London, is a highly acclaimed British-American filmmaker known for his intricate storytelling and visionary approach to cinema. Rising to fame with his mind-bending films like "Memento" (2000) and "Inception" (2010), Nolan gained worldwide recognition for revitalizing the superhero genre with his Batman trilogy, including "Batman Begins" (2005), "The Dark Knight" (2008), and "The Dark Knight Rises" (2012). His works are characterized by complex narratives, non-linear storytelling, and deep exploration of themes like memory, time, and human psychology. Nolan's ability to blend intellectual depth with mainstream appeal has solidified his reputation as one of the most influential directors in contemporary filmmaking."}
Register new user /users POST A JSON user object. Example: {username, email, password, favorite movies} JSON object with user details. Example:{ username: alexbohush, password: xxxxx1234!, email: alex.bohush@gmail.com, favorite movies: []}
Update user info /users/:userId PUT A JSON user object {username, email, password, favorite movies} JSON object with updated user details
Allow users to add movie to list of favorites //users/:userId/favorites POST Movie object {title, genre, director, imageUrl} JSON object indicating movie added
Remove movie from favorites /users/:userId/favorites/:movieId DELETE None JSON object indicating movie removed
Deregister user /users/:userId/ DELETE None A text message indicating successful/unsuccessful removal of user
